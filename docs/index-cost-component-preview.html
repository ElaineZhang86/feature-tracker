Column order: Item Name · Qty · UOM · Unit Cost (Actual) · Index Cost · Markup % · Unit Price · Overflow

ColumnNotes
Unit Cost (Actual)Always shown. Read-only & grayed when item has a catalogue entry. Tooltip: "Actual purchase/inventory cost. Used for job costing only."
Index Cost NewShows CostCatalogueEntry.unitCost for matched items, — for unmatched. Read-only. Tooltip: "Pricing basis from your Cost Catalogue. Markup is applied to this value."
Unit PriceCalculated from Index Cost (not Unit Cost) when catalogue entry exists
Conditional renderBoth new columns hidden entirely when FF off or job has no costCatalogueId
